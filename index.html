NBA Player/Team Search Engine

My program is written in Python that uses both the NBA_API and Qwant Search Engine API. This program takes NBA player and team search queries to provide the user with up-to-date stats and information along with a photo/logo of the respective player/team.


Future Update Features

  • Player Comparisons
